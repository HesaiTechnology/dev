角度同步（Sync Angle）

概述​

禾赛激光雷达产品支持角度同步（Sync Angle）功能，即允许激光雷达在整秒时刻转动至指定的水平角度位置。这一功能对于实现激光雷达与其他设备（如相机、毫米波雷达和其他激光雷达）的实时角度和数据同步起到了关键作用，可以有效保障多传感器感知系统的高精度和稳定性。

原理​

基于激光雷达水平角度位置的定义（图1），激光雷达通过对其电机的转速和相位进行实时闭环控制，以确保在每个整秒时刻到来时，其水平角度都会指向预设的角度方向，以达到与参考信号的同步。考虑到禾赛激光雷达的帧率配置，实际激光雷达工作时会以每整百ms（10Hz）或整50ms（20Hz）为周期转向预期的角度，确保每一帧的数据均能实现角度同步。

在包含激光雷达的多传感器系统中，实现角度同步的重要前提是这些传感器必须通过同一个外部时钟源进行时间同步。 对于禾赛激光雷达而言，可以采用以下两种方式进行时间同步：

  • PTP时间同步协议：此同步方式对应的整秒时刻定义为PTP主时钟发出的整秒时间戳；
  • GPS/PPS时间同步：此同步方式对应的整秒时刻定义为外部输入雷达的PPS（即秒脉冲）信号的上升沿（仅部分禾赛激光雷达产品支持GPS时间同步功能）；

注：若在未进行时间同步的状态下启用了该功能，禾赛激光雷达仍将基于其内部时钟PPS 1Hz信号的上升沿对自身输出的水平角度进行同步控制，但由于不同传感器之间内部时钟的差异，无法实现相互之间的准确角度同步。

实现方法​

角度同步主要通过以下两个步骤实现：

  1. 时间同步：首先，所有需要同步的激光雷达传感器都必须从外部时钟源进行时间同步，这可以通过前面提到的GPS或PTP的方式实现；
  2. 参数配置：一旦传感器完成时间同步，就需要对有关参数进行配置，包括是否启用角度同步和同步角设定值，此设定角度明确了激光雷达在整秒时刻应该指向的预定角度。禾赛激光雷达产品支持以下几种方式对此参数进行配置：

应用案例​

角度同步在现实中多个应用场景中都发挥着作用。

案例1 多传感器角度同步​

在自动驾驶方案中，通常需要同一台车上的多个激光雷达传感器能够同时指向车的前方，以确保它们能够捕捉到相同的环境信息（如图2所示）；然而由于这些激光雷达通常安装在不同位置，相应的硬件接线也指向不同的方向。此时通过对这些雷达启用角度同步并设置相应的角度值，可以确保这些激光雷达在时间同步后，在相同的时刻指向相同的方向，从而提高数据的准确性和一致性。

案例2 激光雷达抗干扰​

参考图3中的情形，当两个激光雷达在车上的安装位置靠得很近时，会有极大概率产生严重的相互干扰，导致出现大量噪点使得输出数据质量下降的问题。

对于这种场景，最佳方案是在这两个激光雷达置物理屏障以从根源上避免相互干扰。但当这种方案不可行时，我们仍然可以通过使用角度同步功能消除此问题：考虑到干扰通常只会在一个激光雷达发射的激光被另一个激光雷达接收时发生，使用角度同步迫使两个激光雷达同时指向对方形成对射，从而使干扰仅发生在的水平角度范围非常有限，且未生成与探测场景强相关的重要数据时，与此同时通过设置激光雷达的有效FOV范围使其在对射区间内不发激光的即可。该方案的具体操作过程如下：

  1. 对两台激光雷达进行时间同步；

  2. 设置同步角度值。结合图3中的案例，当这两台雷达的接线方向一致，即X轴对应的水平0°均指向车的正后方时，可以将雷达#1（左侧）的同步角设为270°，雷达#2（右侧）的同步角设为90°，由此可实现两台雷达的激光在同一时刻均射向对方；
