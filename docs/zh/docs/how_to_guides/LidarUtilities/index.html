使用 LidarUtilities

1 概述​

LidarUtilities 是禾赛激光雷达产品的上位机控制软件，可用于读取激光雷达型号、版本和运行状态等信息，并支持参数配置和软固件升级等功能。

2 使用方法​

2.1 下载与安装​

目前最新版本 PandarView2 (v2.1.8) 中已集成了 LidarUtilities 软件，下载最新版本的 PandarView，可同步安装 LidarUtilities。

注：若通过上述链接下载失败，请联系 Hesai 技术人员获取。

点击 [Tool]，选择 [Lidar Utilities]。

在使用 LidarUtilities 连接雷达前，需要先确认：

  • 激光雷达可以正常工作；

  • 电脑主机能正常接收点云 UDP 数据包，可通过 wireshark 软件检查确认。

2.2 启动与连接​

2.2.1 以太网连接模式​

  1. 参考如何建立以太网连接 中所述方法，设置接收主机的 IP 地址。

  2. 在 LidarUtilities 主界面最上方输入激光雷达的源 IP 地址（IPv4 Address）及 PTC/PTCS 端口号。（默认 IPv4 Address：192.168.1.201，PTC/PTCS 端口号：9347）。

  3. 选择使用的产品型号，点击 [connect]。

  4. 连接成功后，产品型号前红色的圆点会变为绿色。

2.2.2 串口连接模式（仅 JT16 支持）​

  1. 参考如何建立串口连接 中所述方法，确认激光雷达所使用的串口号。

  2. 在 LidarUtilities 主界面最上方输入激光雷达的 RS485 串口号、UART 串口号和波特率（3125000/3000000）。

  3. 选择使用的产品型号，点击 [connect]。

  4. 连接成功后，产品型号前红色的圆点会变为绿色。

2.3 用户界面显示​

成功连接激光雷达后的 LidarUtilities 主界面如下图所示：

  • 目前 LidarUtilities 支持的激光雷达型号为：AT128P、AT360、ATX、JT128、JT16、OT128、FT120、FTX、ETX。其它型号的禾赛激光雷达产品可直接在网页端进行连接和配置。

注：以下图例均以 AT128P 为例，其它型号产品在 LidarUtilities 界面中显示的参数信息会有所差异，具体可查看对应型号的用户手册。

2.3.1 Lidar 信息​

SN：设备序列号

PN：零件号

Software Version：软件版本号

Firmware Version：固件版本号

2.3.2 网络连接参数​

Source IPv4 Address：雷达源 IPv4 地址（默认：192.168.1.201）

IPv4 Mask：子网掩码（默认：255.255.255.0）

IPv4 Gateway：网关 IP 地址（默认：192.168.1.1）

VLAN ID：vlan 开关及 vlan id 设置 （默认：关）

MAC Address：雷达 MAC 地址，不支持修改

Destination IPv4 Address：点云 UDP 的目的端 IPv4 地址 （默认：255.255.255.255）

UDP Port：点云 UDP 端口号（默认：2368）

Fault Message Destination Port：故障信息 UDP 端口号（默认：2368）

2.3.3 Lidar 功能​

Return Mode：回波模式，单回波：最强/最后/第一；双回波：最后及最强/第一及最强

Sync Angle：同步角度，每当整秒时刻，雷达将转动到该角度位置 （默认：关）

Trigger Method：激光发光的触发方式（默认角度触发，可选时间触发）

Reflectivity Mapping：反射率映射模式（默认线性映射，可选非线性映射）

Standby Mode：运行/待机模式（待机模式下，电机停转且激光雷达不发光）

Spin Rate (RPM)：电机转速（默认 200RPM，可选 400RPM）

Rotation Direction：电机旋转方向，以俯视图为准（默认顺时针，可选逆时针）

2.3.4 时间同步​

PTP Status：PTP 状态

PTP 状态描述
Free run雷达未接收到任何 PTP 报文，使用本地时钟自由运行，未进行时间同步
Frozen雷达曾经同步成功，但长时间未收到有效 PTP 报文，时钟保持在最后的同步状态并停止更新
Locked雷达已成功与 PTP 主时钟同步，时间已精确对齐，处于稳定同步状态
Tracking雷达已开始接收 PTP 报文，正在尝试与主时钟同步，但尚未完全锁定

PTP Profile：PTP 同步协议 （802.1AS / 802.1AS Automotive）

PTP Lock Time Offset：PTP 锁定状态时间偏移上限阈值 （默认：1 微秒）

PTP Domain Number：PTP 本地时钟的域序列号（默认：0）

PTP Network Transport：PTP 网络传输协议 （默认：L2）

Switch Type：网络交换机类型 （TSN / Non-TSN）

2.3.5 运行状态统计​

“运行状态”页面用于查看激光雷达的关键工作状态，包括：

Internal Temperature：显示雷达当前机内温度，用于判断设备是否处于正常散热范围。

Input Current：输入电流（在激光雷达的连接器入口处测量）

Input Voltage：输入电压（在激光雷达的连接器入口处测量）

Input Power：输入功耗（在激光雷达的连接器入口处测量）

Start-Up Times：雷达启动次数

System Uptime：本次上电运行时间

Total Operation Time：总运行时间

2.3.6 软固件升级​

  1. 准备工作

  • 请联系禾赛技术支持以获取升级文件。

  • 在升级过程中，建议在雷达光学窗口上覆盖防护罩（随雷达提供）或其他不透光材料。

  1. 升级步骤
  • 点击 [Upload] 按钮并选择升级文件。
  1. 升级完成后

  • 点击右上角的 [Reboot Lidar]，或重新连接雷达。

  • 刷新页面后，新版本号将显示在基本信息区域。

2.3.7 日志​

此页面用于快速查看激光雷达内部记录的日志，内容包括雷达连接状态、BytesFlow（数据流量）开启/关闭、雷达型号切换提示、错误与警告信息以及固件升级过程的详细输出信息，便于禾赛内部快速定位并分析故障原因。

2.3.8 发送 PTC 指令​

此页面提供了 TCP 控制接口，便于用户直接向激光雷达发送 TCP API 指令，具体方法详见使用 LidarUtilities 发送 TCP API 指令 。
